# Стринги Алисы - Портативный Яндекс Браузер без телеметрии

<img width="1920" height="1037" alt="preview" src="https://github.com/user-attachments/assets/02515777-15a2-42d5-8695-29564b6183b0" />

**Полностью портативная версия Яндекс Браузера с отключением телеметрии, служб обновления и фоновой активности.**
> ⚠️ **Внимание:** Это неофициальная модификация. Все права на оригинальный браузер принадлежат Яндекс.

Эта сборка — результат бескомпромиссной войны с телеметрией, фоновыми службами и скрытыми слежками Яндекс Браузера. Здесь нет ничего лишнего. Только браузер. Без докладов «наверх».

## 📥 УСТАНОВКА

### Шаг 1: Установите оригинальный Яндекс Браузер

Скачайте и установите Яндекс Браузер с официального сайта:
- [browser.yandex.ru](https://browser.yandex.ru/)

**Почему это нужно?** Сборка использует оригинальные файлы браузера. Мы не распространяем их по лицензионным причинам.

### Шаг 2: Скачайте сборку Alices Thong

```bash
# Скачайте архив с релиза
https://github.com/anton18-png/Alices-thong/releases/download/v2/Alices-Thong.7z
```

### Шаг 3: Запустите установку

1. **Распакуйте архив** в любую папку (например, `C:\Apps\Alices-Thong\` или на флешку)

2. **Запустите `install.bat`** от имени администратора

```batch
install.bat
```

**Что делает установщик:**

```batch
@echo off
setlocal

rem Задайте путь к исходной папке Яндекс Браузера
set "sourceFolder=%USERPROFILE%\AppData\Local\Yandex\YandexBrowser\Application"

rem Путь назначения (текущая папка)
set "destFolder=%cd%\app"

rem Создаем целевую папку, если её нет
if not exist "%destFolder%" (
    mkdir "%destFolder%"
)

rem Копируем все файлы браузера
xcopy "%sourceFolder%\*" "%destFolder%\" /S /E /Y /H

rem Запускаем очистку от телеметрии
cd app
CleanYandex.bat
```

**Что произойдёт после запуска:**

| Действие | Описание |
|----------|----------|
| 📁 Копирование | Все файлы Яндекс Браузера копируются в папку `app` |
| 🧹 Очистка | Скрипт `CleanYandex.bat` удаляет телеметрию, `.sig` файлы и мусор |
| 🚀 Запуск | Автоматически стартует `StartAlicesThong.exe` |

### Шаг 4: Готово!

Теперь у вас есть **полностью портативная версия** Яндекс Браузера. Её можно:

- 💾 **Перенести на флешку** — запускать на любом компьютере
- 🗑️ **Удалить оригинальный Яндекс Браузер** — сборка работает независимо
- 📦 **Скопировать в другую папку** — просто переместите всю папку

## 🗡️ АРСЕНАЛ УНИЧТОЖЕНИЯ ТЕЛЕМЕТРИИ

### 1. УДАЛЕНИЕ СЛУЖБ (sc delete)

```batch
YandexBrowserService      # Основная служба
YandexUpdaterService      # Обновления
YaTransmgr                # Транспортный менеджер
YandexProtectService      # "Защита" (читай — слежка)
YandexDiskSync            # Синхронизация с диском
YandexBrowserCrashHandler # Отправка крашей
```

**Почему это важно:** Эти службы живут в системе даже после закрытия браузера. Они отправляют телеметрию, проверяют обновления и следят за тобой. Мы их **останавливаем, отключаем и удаляем**.

### 2. ЗАЧИСТКА ПЛАНИРОВЩИКА (schtasks /delete)

```batch
YandexUpdaterTask
YandexProtectTask
YandexBrowserUpdaterTask
YandexUpdateTaskMachineCore
YandexUpdateTaskMachineUA
```

**Почему это важно:** Яндекс умеет восстанавливать свои службы через задания планировщика. Мы вырываем это с корнем.

### 3. БЛОКИРОВКА ЧЕРЕЗ HOSTS

```batch
127.0.0.1 update.api.yandex.net     # Обновления
127.0.0.1 events.yandex.net         # События
127.0.0.1 metrics.yandex.com        # Метрики
127.0.0.1 api-metrics.yandex.net    # API метрик
127.0.0.1 browser-update.yandex.net # Обновления браузера
```

**Почему это важно:** Даже если браузер попытается что-то отправить — данные уйдут в пустоту (на localhost).

### 4. ПОЛИТИКИ РЕЕСТРА (HKLM\SOFTWARE\Policies\Yandex\YandexBrowser)

| Ключ | Значение | Эффект |
|------|----------|--------|
| `BackgroundModeEnabled` | 0 | Браузер не висит в фоне |
| `ScreenshotsEnabled` | 0 | Нельзя делать скриншоты |
| `RestoreOnStartup` | 4 | Новая вкладка при старте |
| `UpdateDefault` | 0 | Автообновления выключены |
| `MetricsReportingEnabled` | 0 | Телеметрия не шлётся |

**Почему это важно:** Политики имеют **высший приоритет**. Браузер НЕ МОЖЕТ их игнорировать.

### 5. ФЛАГИ ЗАПУСКА (через cmd)

```batch
--disable-background-mode      # Запрет фоновой работы
--disable-features=Screenshots # Отключение скриншотов
--restore-on-startup=0         # Не восстанавливать вкладки
--disable-usage-statistics     # Отключить статистику
--disable-crash-reporter       # Отключить отправку крашей
--disable-metrics              # Отключить метрики
--disable-sync                 # Отключить синхронизацию
--disable-rlz                  # Отключить маркетинговые метки
```

**Почему это важно:** Флаги переопределяют любые настройки внутри браузера.

## 🔐 ЧТО ТАКОЕ `.sig` ФАЙЛЫ

`.sig` — это **цифровая подпись** для `browser.exe` и `browser.dll`.

| Вопрос | Ответ |
|--------|-------|
| **Что это?** | Электронная подпись Яндекса |
| **Нужны для работы?** | НЕТ |
| **Что делают?** | Проверяют, не изменён ли файл |
| **Можно удалить?** | ДА |
| **Влияют на телеметрию?** | НЕТ |

**Вердикт:** Удаляй без зазрения совести. Они занимают место и не нужны.

## 🎯 EASYLIST — ЧТО ЭТО И ЗАЧЕМ УДАЛЯТЬ

**EasyList** — это самый популярный список фильтров рекламы в мире.

### Как это работает:

```
Браузер загружает страницу → Проверяет по EasyList → Блокирует рекламу
```

### В Яндекс Браузере используется `RuAdList+EasyList`:

- **EasyList** — блокировка международной рекламы
- **RU AdList** — блокировка русскоязычной рекламы (Яндекс, ВК)

### Почему мы их удаляем?

1. **Ты используешь uBlock Origin** — он делает то же самое, но лучше
2. **Экономия места** — ~50-100 МБ
3. **Нет конфликтов** — два блокировщика могут мешать друг другу

### Если удалить:

- Браузер перестанет блокировать рекламу "из коробки"
- **НО** uBlock Origin продолжит работать
- Страницы будут загружаться быстрее
