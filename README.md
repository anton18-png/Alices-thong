## ✨ Особенности

### 🛡️ **Безопасность и приватность**
- ✅ Полное отключение телеметрии и сбора данных
- ✅ Блокировка всех фоновых служб Яндекс
- ✅ Удаление заданий планировщика
- ✅ Блокировка телеметрических доменов через hosts
- ✅ Очистка metricsid (уникального идентификатора)

### ⚡ **Производительность**
- ✅ Отключена работа браузера в фоне после закрытия
- ✅ Отключены автоматические обновления
- ✅ Убран автозапуск при старте Windows

### 🎯 **Удобство**
- ✅ Полностью портативная версия — работает с флешки
- ✅ Кастомная стартовая страница
- ✅ Сохранение данных в папке `data`

## 🎯 Чем отличается от первой версии

| Характеристика | Версия 1.x | Версия 2.0 |
|----------------|------------|------------|
| Основа | Установщик InnoSetup | Портативный лаунчер от portapps |
| Требует установки Яндекса | ✅ Да | ❌ Нет |
| Работает с флешки | ✅ Частично | ✅ Да |
| Данные пользователя | В USER_DATA | В папке `data` |
| Размер установщика | ~1.5 МБ | ~200 МБ |
| Зависимость от системы | Высокая | Минимальная |

## 🔧 Компоненты сборки

### `AlicesThong.exe`
**Назначение:** Портативный лаунчер
- Обеспечивает портативность
- Автоматически сохраняет данные в папку `data`
- Не требует установки

### `StartAlicesThong.exe`
**Назначение:** Скомпилированная версия основного скрипта
- Запускает процесс очистки
- Запускает браузер с флагами приватности
- Экспортирует настройки после закрытия
- Очищает реестр от следов

### `CleanYandex.bat`
**Назначение:** Скрипт для удаления лишних файлов
- Удаляет `.sig` файлы (цифровые подписи)
- Удаляет телеметрические компоненты
- Опционально удаляет стандартные фильтры рекламы

## ⚙️ Что делает скрипт

| Действие | Описание |
|----------|----------|
| **Остановка служб** | Останавливает YandexBrowserService, YandexUpdaterService и др. |
| **Удаление планировщика** | Удаляет задачи YandexUpdaterTask, YandexProtectTask |
| **Применение политик** | Отключает фоновый режим, скриншоты, восстановление вкладок |
| **Редактирование Preferences** | Правит файл настроек браузера |
| **Очистка сессий** | Удаляет временные файлы вкладок |
| **Блокировка hosts** | Блокирует телеметрические домены |
| **Запуск браузера** | Запускает с флагами приватности |
| **Экспорт настроек** | Сохраняет настройки в AlicesThong.reg |
| **Очистка реестра** | Удаляет следы после закрытия |

## 🚀 Флаги запуска

```bash
--allow-outdated-plugins      # Разрешить старые плагины
--disable-logging              # Отключить логи
--disable-breakpad             # Отключить отчёты о крашах
--disable-encryption           # Отключить шифрование
--disable-machine-id           # Отключить ID машины
--disable-usage-statistics     # Отключить статистику
--disable-crash-reporter       # Отключить репортёр крашей
--disable-metrics              # Отключить метрики
--disable-sync                 # Отключить синхронизацию
--disable-rlz                  # Отключить маркетинговые метки
--disable-background-mode      # Отключить фоновый режим
--disable-features=Screenshots # Отключить скриншоты
--restore-on-startup=0         # Не восстанавливать вкладки
```
